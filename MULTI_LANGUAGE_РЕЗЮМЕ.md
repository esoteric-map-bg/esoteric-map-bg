### **СКОРОСТ:**

- **БЕЗ забавяне!**

### **КОД:**
```python
def t(key, **kwargs):
    """Translation helper"""
    lang = st.session_state.get('language', 'bg')
    trans = load_translations(lang)
    return trans[key].format(**kwargs)

# ИЗПОЛЗВАНЕ:
st.title(t('app.title'))
```
